Built for productivity

No extensions to install, no cloud syncing accounts to manage, no lag. Soryq ships with premium defaults.

Multi-pane PTY Terminal

Full PTY shell sessions using xterm.js. Split your view into up to 4 responsive shell panes. Detects your shell automatically.

CodeMirror Code Editor

CodeMirror 6 core featuring syntax highlighting for 15+ languages, built-in Vim mode, minimap, formatting, and formatting on save.

Live Preview Proxy

Local dev server HTTP proxy routing that automatically injects hot-reloading scripts and the built-in DOM inspector.

Git Integration

Trace diffs, review staged/unstaged changes, and commit files in milliseconds without interrupting your workflow.
